What is css?

cascading style sheets (css) is udse to format layout of a webpage


with css, you can control the color, font, the size of the text, the spacing between elements, how elements are positloned and laid out, what background images backgroud color are to be used, different displays for different devices and screen sizes, and much more!


Tip: the wwork cascading means that a style applied to a parent element will also apply to all children elemant within the parent. So, if you set the color of the body text to "blue,"all headings, paragraphs, and other text element with the body will also get same color (unless you specify something else)!
